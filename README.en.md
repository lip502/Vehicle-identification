# 基于MATLAB运动车辆检测（每辆车速度+GUI+总数+注释详细）

#### Description
该课题为基于帧差法的运动车辆目标跟踪，配备一个GUI界面，可实现统计车辆数量，运动轨迹，和统计车辆数量和车流量以及速度等。后期可开发成判断道路是否拥挤等课题。可定制关于违章检测，车辆车型，车标等识别。
